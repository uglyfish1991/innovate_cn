Welcome to my HTML Recap

HTML is typically structured as:

<tagname>Content</tagname>

We use the tags to declare to the browser how we want the content to be displayed, and what properties we want it to have. The browser then renders that content with those properties.

Some elements are standalone elements, or self-closing. They don't follow the tag content endtag rules. These include things like images, line breaks, inputs and areas.
